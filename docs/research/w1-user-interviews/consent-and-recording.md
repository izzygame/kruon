# 访谈授权与录制说明

## 主持人口头说明

本次研究用于改进 Kruon 的本地代理任务管理体验，预计 45 分钟。我们可能记录屏幕、声音和访谈笔记；材料仅用于产品研究和内部决策，不要求展示真实密钥、客户数据或私有代码。参与完全自愿，可以拒答、暂停或要求删除录制。

## 必须逐项确认

- [ ] 参与者同意参加访谈。
- [ ] 参与者同意记录文字笔记。
- [ ] 参与者同意录制声音；若不同意则关闭声音录制。
- [ ] 参与者同意录制屏幕；若不同意则改用虚构夹具演示。
- [ ] 已说明保存期限、访问人员和删除申请渠道。
- [ ] 已提醒关闭通知并移除密钥、客户数据、个人目录和无关窗口。
- [ ] 是否允许去标识化引用：允许 / 不允许。
