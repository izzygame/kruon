# W1 目标用户访谈包

目标：访谈 3～5 名需要在本地使用多个编码代理、需要审批或追溯任务过程的开发者/技术负责人。

材料包括：

- `interview-guide.md`：45 分钟访谈脚本。
- `consent-and-recording.md`：授权说明与确认记录。
- `workflow-recording-template.md`：流程录屏任务和观察表。
- `synthesis-template.md`：跨访谈归纳工作流、审批痛点和 MVP 修订项。

产品方负责招募参与者、确认授权并录制。Codex 在收到材料后负责去标识化归纳。未取得明确授权时不录制，不把仓库内容、密钥、客户数据或个人路径纳入材料。
